# Snake

This is a very simple Snake game.

Controls:
W = Move Up
A = Move Left
S = Move Down
D = Move Left

You will play as a snake and will try to eat as many circles as possible.

Your body will grow by one every time you eat the circles, and your score will add one to it.

If you hit either the outside border or hit the snakes body, you lose!

Try to get the highest score possible!


Shape classes in the `SwingShapes` package came from here: [https://github.com/SER120L/SwingShapes](https://github.com/SER120L/SwingShapes).
